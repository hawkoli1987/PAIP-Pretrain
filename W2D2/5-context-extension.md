## Context length extension

Extend context length as a **separate, final CPT phase**, not mixed into the main domain adaptation. Code Llama (arXiv:2308.12950) extended from 4,096 to 16,384 tokens by modifying RoPE base frequency from θ=10,000 to θ=1,000,000. **YaRN** (Peng et al., arXiv:2309.00071, 2023) is the most widely adopted extension method—it combines NTK-by-parts scaling with attention temperature adjustment and extended Llama 2 13B to 128K context with only 400 training steps. **LongRoPE2** (ICML 2025) extended Llama-3-8B to 128K with only 10B tokens (80× fewer than Meta's approach) using evolutionary search plus mixed context window training. Changing sequence length affects batch size (memory constraints force fewer sequences per batch), total token throughput, and potentially optimal LR.
